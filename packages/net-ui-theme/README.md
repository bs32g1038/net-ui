# NET-UI THEME

[![NPM][npm-version-image]][npm-version-url]

`NET-UI THEME` is an ui component library based on `Flexbox`, be used for [NET-UI](https://at.aotu.io/)

## Install

```bash
npm install net-ui-theme
```

## Usage

```js
import 'net-ui-theme'
```
